## Codebase Patterns
- Project uses Expo SDK 55 with managed workflow

- react-native-reanimated v4 for animations
- TypeScript strict mode enabled
- Entry point: index.ts → expo-router
- API keys via .env with EXPO_PUBLIC_ prefix (process.env.EXPO_PUBLIC_*)
- Dark theme: background #1a1a2e, text #fff
- Screen files: app/index.tsx, app/text2.tsx, app/mixing.tsx, app/result.tsx
